2D Heat Equation Simulation

This application simulates 2D heat diffusion over time using a finite difference method for solving the heat equation. The rectangular simulation domain allows for user-defined boundary conditions and material properties.

Project Architecture:

  • Frontend: Built with Angular, it provides an interactive interface for parameter input and real-time heatmap visualization.
  • Backend: Developed using FastAPI, it performs numerical simulations and handles HTTP requests efficiently.
  • Computation: NumPy is used for matrix operations and implementing the finite difference scheme for temperature updates.
  • Visualization: Matplotlib generates high-resolution heatmaps that are encoded and sent to the frontend.
  • Communication: The frontend and backend communicate via RESTful APIs using JSON payloads.

This simulation platform demonstrates the power of combining scientific Python tools with modern frontend frameworks to create responsive and educational scientific applications.
