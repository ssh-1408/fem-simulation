2D Physics Simulation Platform

This application simulates 2D heat diffusion and 2D fluid flow using finite difference methods for solving partial differential equations over a rectangular domain. Users can define boundary conditions and material properties for each type of simulation.

Project Architecture:

  • Frontend: Built with Angular, it provides an interactive interface for input and real-time visualizations.
  • Backend: Developed using FastAPI to handle computational requests efficiently.
  • Computation: NumPy is used for numerical operations and iterative solvers.
  • Visualization: Matplotlib generates high-resolution images of simulation states.
  • Communication: JSON-based RESTful APIs connect frontend to backend.

This platform demonstrates how scientific computing tools in Python can be combined with modern web technologies for educational and research purposes.
